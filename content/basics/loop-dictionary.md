```python
d = {'person': 2, 'cat': 4, 'spider': 8}
```


```python
for animal in d:
    legs = d[animal]
    print ('A %s has %d legs' % (animal, legs))
```

    A person has 2 legs
    A cat has 4 legs
    A spider has 8 legs
